About Me

I’m a computer science student who enjoys building interactive software that combines logical problem-solving with clean, intuitive user experiences. I’m especially interested in turning abstract ideas into working applications, whether that’s through games, algorithms, or web-based tools.

Through personal and academic projects, I’ve worked with C++, Python, and modern web technologies, focusing on object-oriented design, readable code, and understanding how systems behave as they grow in complexity. I learn best by building things end-to-end and iterating on them over time.

Python pygame Backtracking

An interactive Sudoku game built with Python and pygame that allows users to play, sketch numbers, and validate moves in real time. The project also includes automated solving logic that can compute solutions to valid boards.
